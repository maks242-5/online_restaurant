# online_restaurant
сучасный онлайн ресторан в которо месть меню, регыстрация,логин,основная очень крутая стоинка, также есть админ панель где можно смотреть забронированные столы,клиенты, добавлять в меню стравы, менять статус или информацию о страве. и еще очень много другого, также подключена csrf защита
