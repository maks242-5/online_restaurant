# Tomato&Co — Онлайн-Ресторан 🍕🍔

Сучасний веб-додаток для автоматизації роботи ресторану, бронювання столів та онлайн-замовлень їжі. Створено на базі Flask та SQLite3.

## 🛠️ Функціонал додатка
* **Система користувачів:** Повна реєстрація, авторизація та керування профілем за допомогою унікальних нікнеймів.
* **Онлайн-меню:** Інтерактивний перегляд страв (піца, бургери, мілкшейки) з детальним описом та актуальними цінами.
* **Кошик та замовлення:** Можливість додавати позиції до кошика, створювати замовлення та відстежувати їхній статус в особистому кабінеті.
* **Панель адміністратора (Admin Dashboard):** Повноцінна адмінка для керування меню (додавання нових страв), моніторингу замовлень, перегляду всіх зареєстрованих клієнтів та керування бронюваннями.
* **Бронювання столів:** Інтелектуальна логіка резервації місць у залі із захистом від повторних бронювань.
* **Безпека:** Реалізовано надійне хешування паролів та глобальний захист від CSRF-атак на всі форми додатку.

## 🚀 Стек технологій
* **Backend:** Python (Flask, Flask-Login для сесій, Flask-WTF)
* **Database:** SQLite3 (структурована реляційна база для страв, замовлень, юзерів та бронювань)
* **Frontend:** HTML5, CSS3 (індивідуальна адаптивна верстка та стилізація елементів інтерфейсу)
